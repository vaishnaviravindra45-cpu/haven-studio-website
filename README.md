# Haven Studio Website

Haven Studio is a photography studio website designed using HTML and CSS.
The website includes a home page, login page, and a clean dashboard layout.

## Features
- Home page with studio information
- Login page UI
- Classic and modern color theme
- Beginner-friendly design

## Technologies Used
- HTML
- CSS
- JavaScript (basic)

## Backend
Backend is implemented using Node.js and Express.
It is included in this repository for reference and can be deployed separately.
